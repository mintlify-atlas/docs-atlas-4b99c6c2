## Installation

<Steps>
  <Step title="Clone the repository">
    First, clone the project repository to your local machine:
    
    ```bash
    git clone https://github.com/usuario/astar-romania-pathfinder.git
    cd astar-romania-pathfinder
    ```
  </Step>

  <Step title="Install dependencies">
    The project requires three Python libraries. Install them using pip:
    
    ```bash
    pip install networkx matplotlib
    ```
    
    <Note>
      The `heapq` module is part of Python's standard library and doesn't need separate installation.
    </Note>
    
    ### Required Dependencies
    - **heapq**: Priority queue for the A* open list (built-in)
    - **networkx**: Graph data structure and manipulation
    - **matplotlib**: Graph visualization
  </Step>

  <Step title="Run the pathfinder">
    Execute the main script to find the optimal path:
    
    ```bash
    python astar_shortest_path_romania.py
    ```
  </Step>
</Steps>

## Expected Output

When you run the script, you'll see the following results in your terminal:

```
Ruta óptima de Fagaras a Bucarest: Arad -> Sibiu -> Rimnicu Vilcea -> Pitesti -> Bucharest
Costo total de la ruta: 418
```

Additionally, a visualization window will open displaying:
- The complete Romanian road network graph
- All cities as labeled nodes
- Road distances on each edge
- The optimal path highlighted in red

<Tip>
  The visualization uses a spring layout algorithm, so node positions may vary slightly between runs, but the graph structure and optimal path remain the same.
</Tip>

## Understanding the Code

### Graph Structure

The Romanian road network is defined as a dictionary where each city maps to its neighbors and distances:

```python
romania_map = {
    'Arad': [('Zerind', 75), ('Sibiu', 140), ('Timisoara', 118)],
    'Sibiu': [('Arad', 140), ('Oradea', 151), ('Fagaras', 99), ('Rimnicu Vilcea', 80)],
    'Pitesti': [('Rimnicu Vilcea', 97), ('Craiova', 138), ('Bucharest', 101)],
    'Bucharest': [('Fagaras', 211), ('Pitesti', 101), ('Giurgiu', 90), ('Urziceni', 85)],
    # ... more cities
}
```

### Heuristic Values

Straight-line distance estimates to Bucharest for each city:

```python
heuristics = {
    'Arad': 366, 'Sibiu': 253, 'Pitesti': 100, 'Bucharest': 0,
    'Zerind': 374, 'Oradea': 380, 'Timisoara': 329,
    # ... more cities
}
```

### Running the A* Algorithm

The core function call that finds the optimal path:

```python
optimal_path, total_cost = a_star_search(
    romania_map,      # The graph
    'Arad',          # Start city
    'Bucharest',     # Goal city
    heuristics       # Heuristic estimates
)
```

## Customization

<Warning>
  When modifying start and goal cities, ensure both cities exist in the `romania_map` dictionary to avoid errors.
</Warning>

You can easily modify the pathfinder to search between different cities by changing the parameters in `astar_shortest_path_romania.py:82`:

```python
# Find path from Timisoara to Bucharest instead
optimal_path, total_cost = a_star_search(romania_map, 'Timisoara', 'Bucharest', heuristics)
```
