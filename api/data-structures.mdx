## Overview

The A* algorithm implementation uses two primary data structures: the graph representation and the heuristic values dictionary. Understanding these structures is essential for implementing custom pathfinding scenarios.

## Graph Structure (romania_map)

### Format

The graph is represented as a Python dictionary where:
- **Keys**: Node names (strings)
- **Values**: Lists of tuples, each containing `(neighbor_name, distance)`

### Type Definition

```python
dict[str, list[tuple[str, int]]]
```

### Creating Custom Graphs

You can create your own graph structure following the same format:

```python
# Simple city network example
my_graph = {
    'CityA': [('CityB', 10), ('CityC', 15)],
    'CityB': [('CityA', 10), ('CityD', 12)],
    'CityC': [('CityA', 15), ('CityD', 10)],
    'CityD': [('CityB', 12), ('CityC', 10)]
}
```

**Important considerations:**
- Ensure bidirectional edges (if A connects to B, B should connect to A)
- Use consistent distance values for bidirectional edges
- All nodes referenced as neighbors must also exist as keys

## Heuristic Values (heuristics)

### Format

The heuristics dictionary maps node names to estimated distances to the goal. For A* to find optimal paths, the heuristic must be **admissible** (never overestimate the actual cost).

### Type Definition

```python
dict[str, int]
```

### Complete Heuristic Data

These values represent straight-line distances to Bucharest:

### Properties

- **Goal node**: Always has heuristic value of 0 (Bucharest in this case)
- **Admissible**: Values never overestimate actual shortest path distance
- **Consistent**: Satisfies triangle inequality h(n) ≤ cost(n,n') + h(n')

- **Complete**: Every node in the graph must have a heuristic value

## Data Validation

Before running A* search, ensure:

1. **Graph completeness**: All neighbor references exist as graph keys
2. **Bidirectional consistency**: Edge weights match in both directions
3. **Heuristic coverage**: Every graph node has a heuristic value
4. **Goal heuristic**: The goal node has h(goal) = 0
5. **Valid nodes**: Start and goal nodes exist in both graph and heuristics

```python
def validate_data(graph, heuristics, start, goal):
    # Check start and goal exist
    assert start in graph, f"Start node '{start}' not in graph"
    assert goal in graph, f"Goal node '{goal}' not in graph"
    
    # Check all graph nodes have heuristics
    for node in graph:
        assert node in heuristics, f"Node '{node}' missing heuristic value"
    
    # Check goal heuristic is 0
    assert heuristics[goal] == 0, f"Goal heuristic must be 0, got {heuristics[goal]}"
    
    # Check neighbor references
    for node, neighbors in graph.items():
        for neighbor, _ in neighbors:
            assert neighbor in graph, f"Neighbor '{neighbor}' not in graph"
    
    print("Data validation passed!")
```
