## Finding Paths Between Different Cities

The Romania map contains 20 cities with various connections. You can find the optimal path between any two cities using the same A* algorithm.

## Available Cities

The Romania map includes these cities:

- Arad, Zerind, Oradea, Sibiu, Timisoara
- Lugoj, Mehadia, Drobeta, Craiova
- Rimnicu Vilcea, Fagaras, Pitesti, Bucharest
- Giurgiu, Urziceni, Hirsova, Eforie
- Vaslui, Iasi, Neamt

## Batch Route Finding

You can modify the script to find multiple routes by replacing the execution code (lines 82-117) with a loop:

```python
# Replace lines 82-117 in astar_shortest_path_romania.py with:

# Define route queries
routes = [
    ('Arad', 'Bucharest'),
    ('Oradea', 'Bucharest'),
    ('Timisoara', 'Bucharest'),
    ('Neamt', 'Bucharest'),
    ('Giurgiu', 'Iasi')
]

# Find all routes
print("=" * 60)
for start, goal in routes:
    path, distance = a_star_search(romania_map, start, goal, heuristics)
    if path:
        print(f"{start} → {goal}")
        print(f"  Path: {' -> '.join(path)}")
        print(f"  Distance: {distance} km")
        print("-" * 60)
    else:
        print(f"No path found from {start} to {goal}")
        print("-" * 60)
```

<Tip>
This is useful for comparing multiple route options or analyzing the entire road network.
</Tip>

## Reverse Routes

Since the Romania map is bidirectional, you can find routes in both directions with identical distances:

<CodeGroup>

```python Bucharest to Arad
# Modify line 82 to search from Bucharest to Arad:
optimal_path, total_cost = a_star_search(romania_map, 'Bucharest', 'Arad', heuristics)
# Output: 418 km (same as Arad to Bucharest)
```

```python Arad to Bucharest
# Modify line 82 to search from Arad to Bucharest:
optimal_path, total_cost = a_star_search(romania_map, 'Arad', 'Bucharest', heuristics)
# Output: 418 km (same distance, reversed path)
```

</CodeGroup>

<Note>
The distance will be identical in both directions, but the path list will be reversed.
</Note>

## No Path Scenarios

The A* algorithm handles disconnected cities gracefully. If you search for a city that doesn't exist or isn't connected, the function returns `(None, float('inf'))`. All 20 cities in the Romania map are connected, so you'll always find a valid path between any two cities in the map.

## Common Route Patterns

### West to East Routes
Routes from western cities to eastern cities often go through Bucharest as a central hub:
- Timisoara → Bucharest → Iasi
- Arad → Bucharest → Vaslui
- Oradea → Bucharest → Hirsova

### Coastal Routes
Routes involving coastal cities (Eforie, Hirsova) typically connect through Urziceni and Bucharest:
- Bucharest → Urziceni → Hirsova → Eforie
- Hirsova → Urziceni → Bucharest → Pitesti → ... → Arad

### Northern Routes
Routes in the northern region may avoid Bucharest:
- Oradea → Zerind → Arad or Oradea → Sibiu (depending on destination)
- Iasi → Vaslui → Urziceni (for eastern connections)
